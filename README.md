# 📝 О проекте
Данный репозиторий представляет собой Backend и API для сервиса мониторинга занятости парковок Parking 2.0.

Это pet-проект, созданный в учебных и демонстрационных целях. Основная цель системы — предоставить пользователю актуальную информацию о загруженности парковок в реальном времени, помогая найти свободное место для автомобиля.

Основные возможности сервиса:

    📊 Мониторинг зон: Отслеживание текущего количества автомобилей на различных парковочных зонах.

    🗺️ Карта парковок: Предоставление списка всех доступных парковок с их статусом (свободно/занято).

    🔄 Обновление данных: API для изменения статуса парковочных мест (имитация въезда/выезда авто).

    📜 История: (опционально, если есть) Хранение истории посещений для аналитики загруженности.

Проект спроектирован с упором на чистоту кода, модульность и соблюдение принципов REST API.

# ⚙️ Технологический стек
В проекте используется современный и быстрый стек технологий:

      -Язык программирования: Python 3.9+
      
      -Web-Framework: FastAPI (высокая производительность, автоматическая документация)
      
      -База данных: PostgreSQL (или SQLite для локальной разработки)
      
      -ORM: SQLAlchemy / [Tortoise ORM] (в зависимости от реализации)
      
      -Миграции: Alembic
      
      -Валидация данных: Pydantic
      
      -Контейнеризация: Docker & Docker Compose
      
      -Web-сервер: Uvicorn (ASGI)


# 🚀 Запуск проекта
Требования

    Установленный Python 3.9+
    
    (Опционально) Docker и Docker Compose
    
Локальный запуск (без Docker)
    
    1)Клонируйте репозиторий
    
    2)Создайте и активируйте виртуальное окружение
    
    3)Установите зависимости: pip install -r requirements.txt
    
    4)Запустите сервер: uvicorn app.main:app --reload
